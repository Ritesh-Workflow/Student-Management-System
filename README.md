# Student-Management-System
A Python-based Student Management System developed using Object-Oriented Programming (OOP), File Handling, and Exception Handling.

Features :

- Add Student
- View All Students
- Search Student
- Update Student Details
- Delete Student
- Store Data Using File Handling
- Handle Invalid Input Using Exception Handling

Technologies Used :

- Python
- Object-Oriented Programming (OOP)
- File Handling
- Exception Handling

How to Run :

1. Download or clone the repository.
2. Open the project in Python.
3. Run the main Python file.
4. Follow the menu options displayed on the screen.

Learning Outcomes :

- Working with Classes and Objects
- Managing Data with File Handling
- Handling Errors with Exception Handling
- Building a Menu-Driven Application
